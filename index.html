Introduction

Python is a high-level, interpreted, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its notable use of significant whitespace. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Installation

Python can be downloaded and installed from the official website python.org. It is available on multiple platforms such as Windows, macOS, and various Linux distributions. Most Linux distros come with Python pre-installed. You can verify the installation by running:

python --version
# or
python3 --version
Syntax and Semantics

Syntax and Semantics Python’s syntax is clean and easy to learn. Here are some key points:

  • Indentation defines blocks of code.
  • Statements do not require semicolons (;).
  • Variables are dynamically typed and do not require explicit declaration.

Example:

if True:
    print("Hello, World!")
Variables and Data Types

Python supports several data types:

  • Numbers: int, float, complex
  • Sequences: list, tuple, range
  • Text: str
  • Mappings: dict
  • Sets: set, frozenset
  • Boolean: bool
  • Binary: bytes, bytearray, memoryview

Example:

x = 10          # int
name = "Alice"  # string
pi = 3.14159    # float
is_valid = True # boolean

Functions

Functions are defined using the def keyword:

def greet(name):
    return f"Hello, {name}!"

print(greet("Bob"))

Functions can have default parameters, variable-length arguments, and keyword arguments.

Modules and Packages

Python modules are files containing Python code. You can import modules using the import statement. Example:

import math
print(math.sqrt(16))
Classes and Object-Oriented Programming

Python supports classes and OOP principles:

class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(f"{self.name} says woof!")

dog = Dog("Buddy")
dog.bark()

Supports inheritance, polymorphism, encapsulation, etc.

Exceptions and Error Handling

Use try, except, else, and finally blocks:

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")
else:
    print("Division successful!")
finally:
    print("This runs no matter what.")
File Handling

Open, read, write, and close files

with open('file.txt', 'r') as file:
    content = file.read()
    print(content)

Decorators

Functions that modify other functions:

def decorator(func):
    def wrapper():
        print("Before function call")
        func()
        print("After function call")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()
Generators and Iterators

Generators produce items one at a time using yield:

def countdown(n):
    while n > 0:
        yield n
        n -= 1

for number in countdown(5):
    print(number)
